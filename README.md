# 🍎 基于计算机视觉的网页端水果智能识别与质检系统 (Fruit-SmartID Pro)

本项目是一个面向日常选购、家庭使用及水果盘点场景的**网页端水果智能识别与智能质检服务系统** 。用户只需上传图片或调用摄像头，系统即可快速盘点水果、估算价格，并基于图像特征进行瑕疵与腐烂检测 。同时，系统一体化提供营养科普、产地应季、推荐吃法与储存建议，全方位解决用户在选购、储存和食用水果时的实际痛点 。

---

## ✨ 核心功能特性

系统基于先进的计算机视觉技术，构建了三大核心业务板块：

1. 📷 智能单品识别与全方位科普 

* **多目标定位识别**：基于 YOLOv8 模型，支持多品类常见水果的精准定位与置信度输出 。
* **五位一体知识图谱展示**：
  * **储藏方式**：常温/冷藏、最佳温度与避光/防潮保鲜技巧 。
  * **营养与禁忌**：热量、糖分、膳食纤维科普，提供空腹禁忌与特殊人群（如糖尿病、减肥人群）食用建议 。
  * **产地应季**：结合系统当前时间，智能判定 `【当季推荐】` 或 `【反季水果】` 。
  * **创意食谱**：支持单果或**多果组合**智能匹配直接食用、鲜榨果汁、水果沙拉等创意吃法 。

2. 📊 批量水果统计与视觉计价（新升级） 

* **一键批量盘点**：支持对“一堆水果”进行单张图片内的批量识别与种类统计 。
* **综合指标输出**：实时输出该批次水果的**总数量**、**平均瑕疵率**、**总重量** 。
* **视觉计价器**：支持结合用户输入的单价，自动计算并渲染 `【预估总价】` 界面，贴近市场采购与零售盘点场景 。

3. 🛡️ 新鲜度评估与腐烂检测（视觉核心） 

* **多维特征捕捉**：提取图像的颜色畸变与表面纹理特征，自动识别水果的**腐烂、黑斑、发霉、皱皮、氧化**等瑕疵 。
* **定量打分质检**：采用“检测腐烂区域 + 特征综合打分”的级联逻辑，最终输出清晰的三个质检等级标签及食用建议 ：
  * 🟢 `【新鲜可食】` 
  * 🟡 `【轻微瑕疵】`（建议尽快食用或切除瑕疵部分） 
  * 🔴 `【已变质勿食】`

## 🛠️ 项目技术栈

项目采用前后端分离的轻量化架构，打通了从底层 CV 质检算法到前端响应式交互的全栈流程 ：

| 模块 | 核心技术 | 说明 |
| --- | --- | --- |
| **前端 (Frontend)** | HTML5 + CSS3 + JavaScript + **Vue3** | 响应式设计，卡片式数据面板，支持电脑/手机双端流畅访问 |
| **后端 (Backend)** | **Python** + **Flask / FastAPI** | 负责构建高并发 RESTful API，承载视觉推理与数据库检索流转 |
| **视觉算法 (CV)** | **YOLOv8** + **OpenCV** | **YOLOv8** 用于种类定位与腐烂区域检测；**OpenCV** 用于图像预处理、轮廓提取与像素面积换算 |
| **数据存储 (DB)** | **JSON** / **SQLite** / MySQL | 轻量级结构化存储，便于快速检索水果百科、科普和食谱字段 |

## 📐 算法工作流简述

1. 批量称重预估算法 

利用 OpenCV 的 `findContours` 算法提取各水果独立轮廓，计算其像素面积 $A_{pixel}$ 。通过引入参照物（如手机、手掌）计算出的物理缩放因子 $k$，换算得到实际尺寸，并结合不同品类水果的平均密度 $\rho$ 估算出单个及总重量： 

$$\text{Estimated Weight} = \sum_{i=1}^{n} \left( \text{Volume}(k \cdot A_{pixel\_i}) \times \rho_{fruit\_i} \right)$$

2. 新鲜度打分级联逻辑 

1. 通过 YOLOv8 的缺陷检测分支分割出水果表面的异常区域（如黑斑、发霉、破损） 。
2. 计算瑕疵区域面积占水果整体轮廓面积的百分比（瑕疵率） 。
3. 结合 OpenCV 计算缺陷区域的 HSV 颜色偏移量（如苹果氧化变褐、发霉变白/绿）进行特征打分 。
4. 综合各项指标归一化输出新鲜度得分，触发对应的质检标签 。

## 🚀 快速开始

### 1. 环境准备

确保本地具有 Python 3.9+ 及 Node.js 环境。

### 2. 后端部署

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows 下请使用 venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### 3. 前端部署

```bash
cd ../frontend
npm install
npm run dev
```

打开浏览器访问控制台提示的本地地址（如 `http://localhost:5173`），即可直接调用本地摄像头或上传多张水果照片，体验完整的**智能盘点与质检**服务 ！
